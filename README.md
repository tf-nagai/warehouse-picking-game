# 🏭 倉庫ピッキングゲーム

ブラウザで遊べる倉庫ピッキングシミュレーションゲームです。  
制限時間内にオーダーをこなして高得点を目指そう！

👉 **[ゲームをプレイする](https://あなたのユーザー名.github.io/warehouse-picking-game/)**

※リンクは公開後にご自身のURLに書き換えてください

---

## 📖 遊び方

### 基本の流れ

1. **オーダーを確認** - 画面上部に必要な商品が表示されます
2. **棚でピック** - 光っている棚に近づいて `SPACE` でピック
3. **出荷口へ納品** - 全て集めたら画面下の出荷口で `SPACE` で納品
4. **繰り返し** - 制限時間内にたくさんのオーダーをこなそう！

### 🚨 緊急オーダー

プレイ中、ランダムで **緊急オーダー** が割り込み発生します！

| 選択 | 効果 |
|------|------|
| **Y: 受ける** | 今のオーダーを破棄して緊急処理。高得点 & 時間+10秒！ |
| **N: 無視** | 現オーダー続行。ただし -5秒 ペナルティ |

**判断がカギ！** 今のオーダーがあと少しなら無視、始めたばかりなら受ける...状況に応じた選択が高得点への道！

## 🎯 操作方法

### キーボード

| キー | 動作 |
|------|------|
| `↑` `↓` `←` `→` / `W` `A` `S` `D` | 移動 |
| `SPACE` | ピック / 納品 |
| `Y` / `N` | 緊急オーダーの受諾 / 拒否 |
| `M` | BGM ON/OFF |

### スマートフォン

- 画面左の十字キーで移動
- 右の緑ボタンでピック/納品
- 緊急時は選択ボタンが表示されます
